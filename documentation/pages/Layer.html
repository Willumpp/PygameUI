Layer Class

Layer(layer_name, parent_layer=None)

Layers are useful for handling groups of UI elements and objects.
This includes hiding entire layers at once - disabling their functionality. This is useful for adding menus that can be revealed or hidden, for example.

Parameters

layer_name (str)
The unique name to call the layer (used for searching for the layer)
parent_layer (Layer)
The parent layer of this layer. Think of a tree-like structure.

Methods

get_objects(copy=True) -> List

Returns a list of all objects stored within the Layer.

copy (bool)
Determine if the list of all objects is returned as a copy or by reference

get_ui_elements(copy=True) -> List

Returns a list of all UI elements stored within the Layer.

copy (bool)
Determine if the list of all UI elements is returned as a copy or by reference

find_ui_element(tags, has_all=False) -> list

Find UI elements with specific tags within the layer.

tags (list of str)
List of tags that will be used within the search
has_all (bool)
Specify if all tags in the tags list is required (true) or at least one (false)

find_object(tags, has_all=False) -> list

Find Object with specific tags within the layer.

tags (list of str)
List of tags that will be used within the search
has_all (bool)
Specify if all tags in the tags list is required (true) or at least one (false)

set_upon_visible_func(func, args=())

Set a function to call when the visibility of the layer is set to true

func (function)
Function to call when the visibility of the layer is set to true
args (tuple)
Arguments to pass to the function when it is called

add_objects(single_obj=None, obj_list=[])

Add objects to the layer.
Objects must contain an update method or a draw method.

single_obj (object of any type)
Specify a singular object to add to the layer. Leave as None to ignore this.
obj_list (list)
Provide a list of objects to add to the layer

add_ui_element(single_element=None, element_list=[])

Add UI elements to the layer.

single_element (UIElement)
Specify a singular UI element to add to the layer. Leave as None to ignore this.
element_list (list of UIElement)
Provide a list of UI elements to add to the layer

clear_objects()

Removes all objects from the layer

clear_ui_elements()

Removes all UI elements from the layer

remove_objects(obj_list=[], tags=[])

Remove specified objects from the layer

obj_list (list)
Provide a list of objects that should be removed form the layer.
tags (list of str)
Remove objects from the layer with any of the given tags

remove_ui_elements(element_list=[], tags=[])

Remove specified UI elements from the layer

element_list (list)
Provide a list of UI elements that should be removed form the layer.
tags (list of str)
Remove UI elements from the layer with any of the given tags

add_child(layer)

Add a child layer to this layer

layer (list)
Layer to add as a child to this layer

set_visible(visible)

Set visibility of the layer (but not child layers)

visible (bool)
Specify if the layer should be visible (true) or not (false)

depth_set_visible(visible, set_self=True)

Set visibility of the layer and all child layers

visible (bool)
Specify if the layer and children should be visible (true) or not (false)
set_self (bool)
Specify if this layer should also be affected (true) or not (false; just the child layers)

update(events)

Call update() on all objects and UI elements in this layer
Call update() on all child layers of this layer also.

events (list)
Event list from pygame.event.get()

draw()

Call draw() on all objects and UI elements in this layer
Call draw() on all child layers of this layer also.

find_layer(layer_name, path=None) -> tuple

Find and return a tuple containing if the layer was successfully found (index 0) and the path to that layer (queue of layers to that layer)

layer_name (str)
The name of the layer to find
path (Queue)
Queue of preceding layers. This should be ignored as its not meant to be assigned upon first call

switch_layer(layer_name, from_parent=False)

Reveals the given layer name and hides all other children

layer_name (str)
The name of the layer to find and reveal
from_parent (bool)
Switch from the parent layer instead of this layer

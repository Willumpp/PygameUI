Text Input

TextInput(surface, xpos, ypos, width, height, default_text, max_characters, use_int=False, max_val=2**32, min_val=-2**32)

Allows for text input into a drawn box

Parent:
UIElement

Parameters

default_text (int)
The text which is displayed when the text box is empty (the text is not retrieved with get_text())
max_characters (int)
The maximum number of characters the textbox holds
use_int (bool)
Determines if the text box can only hold numbers
max_val (bool)
If use_int is true, represents the maximum number the box can hold
min_val (bool)
If use_int is true, represents the minimum number the box can hold

Methods

has_text() -> bool

Returns if the textbox has text within

get_text() -> str or int

Returns the text typed within the text box.
If use_int is true, returns an integer. Otherwise, return a string of the typed text.

update(events)

Provides clicking detection for the text input UI element.
self.visible must be true to trigger.
Additionally provides key detection for typing.

events (bool)
Event list from pygame.event.get()

draw()

Draws background, text, border (respective order)
self.visible must be true to trigger.

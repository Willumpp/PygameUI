Buttons

Button(surface, xpos, ypos, width, height, function, args=())

Draws a button which can be pressed and call a given function with specified arguments

Parent:
UIElement

Parameters

function (function)
Function called when the button is pressed
args (tuple)
Parameters for the function to use when called

Methods

force_event()

Forces and triggers a button press

draw()

Draws background, sprite, text, border (respective order)
self.visible must be true to trigger.

update(events)

Provides clicking detection for the button UI element.
self.visible must be true to trigger.

events (bool)
Event list from pygame.event.get()

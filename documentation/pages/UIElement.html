UIElement Parent Class

UIElement(surface, xpos, ypos, width, height, font="Comic Sans MS", font_size=14)

All UI elements inherit from this class and should share its functionality.
Inherit from this to create custom UI elements.

Parameters

surface (pygame surface)
The pygame surface for the TextBox to be drawn on
xpos (int)
X-coordinate of the UI element. Origin at top left of surface
ypos (int)
Y-coordinate of the UI element. Origin at top left of surface
width (int)
Pixel width of the UI element
height (int)
Pixel height of the UI element
font (str)
Font family for the drawn text
font_size (int)
Size of the drawn text

Attributes

bgcol (tuple of int)
Background colour to fill in _draw_background()
fgcol (tuple of int)
Foreground colour. Mainly to draw text and border.
draw_border (bool)
Determines if a border should be drawn in _draw_border()

Methods

get_pos() -> Vector

Returns the Vector position (surface x and y coordinate) of the UI element.

set_pos(xpos, ypos)

Sets the position of the UI element on it's surface.

xpos (int)
Surface x-position for the UI element
ypos (int)
Surface y-position for the UI element

set_visible(active)

Sets the visibility of the UI element.
Use decorator @require_visibility to make set_visible() actually useful.

visible (bool)
Value (True/False) to set the self.visible attribute.

get_bbox() -> Vector

Returns the bounding box position of the UI element.
Bounding box position is the bottom left corner Vector position of the UI element.

collision_point(pos) -> Bool

Returns if the given position is within the bounding area of the UI element.

pos (Vector)
The point position to test if within the bounding area

set_image(image_dir, width=0, height=0) -> Bool

Sets the image (sprite) for the UI element to draw.
Use self._draw_sprite() within the self.draw() method to make self.set_image() actually useful.

image_dir (str)
Directory of the image file. (must be accepted by pygame).
width (int)
Pixel width of the image. If set to 0, width is set to the UI element's width.
height (int)
Pixel height of the image. If set to 0, height is set to the UI element's height.

update(events=[])

Empty in parent class.
Meant to be called every frame for interacivity with a UI element.

events (list)
Event list from pygame.event.get(). Used for checking for mouse and key presses (as well as other things).

draw()

Empty in parent class.
Meant to be called every frame and include pygame functions to draw the UI element to the screen.
These functions are also relevant:

  • self._draw_background()
  • self._draw_text()
  • self._draw_border()
  • self._draw_sprite()

set_tags(tags)

Replaces all tags in the UI element with the tags provided.

tags (list)
List of tags (strings) to give to the UI element.
This list is copied and unedited.

get_tags() -> list

Gets the list of tags (strings) from the UI element.

add_tag(tag)

Appends the given tag to the tag list of the UI element.

tag (str)
Tag to append to the tag list

remove_tag(tag)

Remove the given tag from the UI element's tag list.

tag (str)
Tag to remove from the tag list

has_tags(tags, needs_all=True) -> bool

Returns if object has all tags in passed tag list.

tags (list)
List of tags that the UI element must contain
needs_all (bool)
Determines if at least one of the tags in the list is needed (false) or all (true)

has_tag(tag) -> bool

Returns if tag is in the UI element's tags

tag (str)
Tag the UI element must contain

set_text(text)

Sets the drawn text of the UI element to the string provided

text (str)
Text to change the UI element's drawn text to

get_text() -> str

Returns the drawn text of the UI element

set_font(font_name, size, align="left", args=())

Sets the font of the UI elements drawn text

font_name (str)
Font family/name of the font to use
size (int)
The size to drawn the given font
align (str)
The font-aling to draw the text.
This can be:
  • "left"
  • "centre"
args (tuple)
Any additional arguments for the pygame.sysfont.SysFont() function.

require_visible(function)

Decorator to only trigger a function if self.visible is true.
Use set_visible() to change the self.visible attribute.

function (function)
Function to only trigger when true

_draw_background()

Fills the UI element's bounding area/size to the colour of self.bgcol
Intended to be used in the self.draw() method.

_draw_text(custom_text="")

Draws text over the UI element.
Use set_text() to change the text which is drawn.
Colour of the text is drawn in self.fgcol

custom_text (str)
Text to draw instead of self._text.
If string is empty, draws text stored in self._text

_draw_border()

Draws text bounding box/collision box.
Colour of the border is drawn in self.fgcol

_draw_sprite()

Draws a sprite to fit the size of the UI element's bounding area.
Set the sprite with set_image()

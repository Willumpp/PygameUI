UIElement Parent Class

UIElement(surface, xpos, ypos, width, height, font="Comic Sans MS", font_size=14)

All UI elements inherit from this class and should share its functionality.
Inherit from this to create custom UI elements.

Parameters

surface
The pygame surface for the TextBox to be drawn on
xpos
X-coordinate of the UI element. Origin at top left of surface
ypos
Y-coordinate of the UI element. Origin at top left of surface
width
Pixel width of the UI element
height
Pixel height of the UI element
font
Font family for the drawn text
font_size
Size of the drawn text

Methods

get_pos() -> Vector

Returns the Vector position (surface x and y coordinate) of the UI element.

set_pos(xpos, ypos)

Sets the position of the UI element on it's surface.

xpos (int)
Surface x-position for the UI element
ypos (int)
Surface y-position for the UI element

set_visible(active)

Sets the visibility of the UI element.
Use decorator @require_visibility to make set_visible() actually useful.

visible (bool)
Value (True/False) to set the self.visible attribute.

get_bbox() -> Vector

Returns the bounding box position of the UI element.
Bounding box position is the bottom left corner Vector position of the UI element.

collision_point(pos) -> Bool

Returns if the given position is within the bounding area of the UI element.

pos (Vector)
The point position to test if within the bounding area

set_image(image_dir, width=0, height=0) -> Bool

Sets the image (sprite) for the UI element to draw.
Use self._draw_sprite() within the self.draw() method to make self.set_image() actually useful.

image_dir (string)
Directory of the image file. (must be accepted by pygame).
width (int)
Pixel width of the image. If set to 0, width is set to the UI element's width.
height (int)
Pixel height of the image. If set to 0, height is set to the UI element's height.

update(events=[])

Empty in parent class.
Meant to be called every frame for interacivity with a UI element.

events (list)
Event list from pygame.event.get(). Used for checking for mouse and key presses (as well as other things).

draw()

Empty in parent class.
Meant to be called every frame and include pygame functions to draw the UI element to the screen.
These functions are also relevant:

  • self._draw_background()
  • self._draw_text()
  • self._draw_border()
  • self._draw_sprite()

set_tags(tags)

Assings multiple tags to the UI element

tags (list)
List of tags (strings) to give to the UI element.
This list is copied and unedited.

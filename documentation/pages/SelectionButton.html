Selection Button

SelectionButton(surface, xpos, ypos, width, height, function, selection_col, default_col, buttons, args=())

Draws a button which can be pressed and call a given function with specified arguments.
This differs from a regular button as it changes the colour of every other button when it is pressed. (like a radio button)

Parent:
Button

Parameters

selection_col (tuple of int)
Background colour to fill in when this button is selected
default_col (tuple of int)
Background colour to fill in when this button is de-selected (when another different selection button is pressed)
buttons (list of SelectionButton)
A list of all selection buttons that will be affected when one is pressed.
